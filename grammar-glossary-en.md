# Grammar Glossary: English (as used in exercises)

## Cases

| Case | Short label | Description |
|---|---|---|
| Nominative | subject | Who is doing it? |
| Accusative | object | What? / Where to? |
| Instrumental | by/with | By whom/what? With whom/what? |
| Dative | to/for | To whom? For whom? For what purpose? |
| Ablative | from | From where? From whom? Because of what? |
| Genitive | of | Of whom? Of what? Whose? |
| Locative | to/at | Where? In what? On what? |
| Vocative | address | Direct address: "Hey, man!" |

## Grammatical concepts

| Term | As used in exercises |
|---|---|
| declension | adding case endings to a stem (gender, number, case) |
| conjugation | inflecting verbs for person and number |
| root | the most basic, irreducible element carrying core meaning (e.g. √budh) |
| prefix | element added before a root (e.g. saṁ-) |
| suffix | element added after a root (e.g. -ta, -ma) |
| stem | root combined with a suffix; the base for inflection |
| ending | case ending added to a stem |
| base | stem form obtained by removing the 3rd.sg. termination -ti |
| case | grammatical relationship of a noun to other words |
| gender | masculine, feminine, or neuter |
| number | singular or plural |
| word order | Subject (nom) + Object (acc) + Verb |
| negation | particle *na* before a verb: "does not" |
| prohibition | particle *mā* before an imperative: "Don't!" |
| attribution | A is B; nominal sentence |
| nominal sentence | subject + attribute, with *hoti* often omitted |
| equational phrase | A is B |
| vowel | a, ā, i, ī, u, ū, e, o |
| accusative of motion | accusative indicating where the subject is going to |
| verbal termination | person/number ending on a verb (-mi, -si, -ti, etc.) |
| honorific plural | 2nd.pl. form used when addressing a senior |
| present tense | actions happening now, occurring regularly, or general truths |
